# Famozo catalog + SQLite

Каталог товаров переведен на локальную базу SQLite.

## Что добавлено
- `database/catalog.db` — база товаров.
- `server.py` — локальный сервер на Python без внешних библиотек.
- `assets/catalog-db.js` — выводит товары из базы в существующую карточку товара.
- `admin.html` — админка для добавления, редактирования и удаления товаров.

## Запуск
```bash
python server.py
```

Каталог: http://127.0.0.1:8000/catalog.html
Админка: http://127.0.0.1:8000/admin.html

Важно: через двойной клик по `catalog.html` база не загрузится. Нужно запускать через `python server.py`.

## Категории через админку

В этой версии категории тоже хранятся в SQLite.

Админка: `http://127.0.0.1:8000/admin.html`

Что можно делать:

- создать новую категорию;
- задать название, большой раздел, картинку и порядок;
- скрыть категорию с сайта через поле `Активна = Нет`;
- добавлять товары в любую созданную категорию;
- менять ключ категории — товары автоматически перенесутся на новый ключ.

Новая активная категория автоматически появится в верхнем блоке ссылок каталога и получит собственный раздел товаров ниже.

Важно: ключ категории пишется латиницей, например `chasy`, `obuv`, `novaya_kategoriya`.


## Проверка правильной версии

В новой версии на странице `admin.html` сверху написано: **Админка каталога — версия с категориями**.
Ниже сразу должен быть блок **Добавить категорию**.

Если этого нет, значит открыт старый проект/старый сервер. Нужно:
1. Полностью остановить сервер через `Ctrl+C`.
2. Распаковать этот архив в новую пустую папку.
3. Запустить `python server.py` именно из этой новой папки.
4. Открыть `http://127.0.0.1:8000/admin.html` и нажать `Ctrl+F5`.
